# Petgram

Petgram is Social Media Network for pets.


## Run
```
    npm install
    npm run dev
```
Starts on `http://localhost:8080`

## Build
```
    npm run build
```

## Development Technologies
* React.js
* Reach Router
* ReacT Icons
* GraphQL & React Apollo 
* Styled-Components
* Lodash
* Webpack
